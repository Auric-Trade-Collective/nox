# Nox (Version 0.0.2)
This is the first alpha version of nox, therefore, it will
have full CLI suppport and webserver features. Although
primitive, it should be robust and useful.

## Added

**Multiple API targets allowed**
This version will allow multiple libraries to be specified
and loaded as APIs. In the toml file they can be used like so:

```toml
api = [ "/api/lib1.so", "/api/lib2.so", "/api/lib3.so" ]
```
**Exposed Logging Functions to APIs and Plugins**
The following definitions have been added to native/webapi.h
and plugin/plugin.h.

```c
#ifndef NOXLOG
#define NOXLOG

void LogWrite(char *name_space, char *msg);
void LogWarn(char *name_space, char *msg);
void LogError(char *name_space, char *msg);
void LogPanic(char *name_space, char *msg);

#endif
```

**Header Files Reorganized Into nox.h**
There is now a single source of truth for all of the Nox C
code.

## Unreleased
