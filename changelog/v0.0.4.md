# Nox (Version 0.0.3)

## Added

**Enhanced Status Codes**
Previously there were only a few custom status codes, and they
were only in the form of custom errors. Now there are custom status
codes and pages. As well as quick response message functions.

Custom status code pages are now defined in ``root_dir/status/``.

```c
void ApiError(HttpResponse *resp, int code, char *message);
void ApiErrorPage(HttpResponse *resp, int code);
```

Error messages given by Nox itself are now enhanced and more descriptive.

## Patched

**DLL Loading Debug Message**
Previously Nox wouldn't say much information if it failed to load a DLL file. Now
it has a debuge message which can be shown with the ``--debug`` flag.
