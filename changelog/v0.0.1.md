# Nox (Version 0.0.1)
This is the first alpha version of nox, therefore, it will
have full CLI suppport and webserver features. Although
primitive, it should be robust and useful.

## Added

**Cookies**
New cookie functions in native/webapi.h
```c
char *TryGetCookie(HttpRequest *req, char *key);
void TrySetCookie(HttpResponse *resp, char *key, char *value, char *path, long expires, bool secure, bool httponly);
```

**Full CLI Support**
Nox's CLI support will be completed with new commands being
added as follows:

```
nox init [name] # Initializes a nox.toml
nox install # Installs nox onto your system
```

**Usage License**
Complete, will be added in version 0.0.1 push

## Unreleased

**Swagger Support**
Nothing yet: likely will be shipped as a closed source enterprise plugin.
